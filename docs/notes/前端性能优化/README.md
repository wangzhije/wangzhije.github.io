## 一次 js 请求一般情况下有哪些地方会有缓存处理

dns 缓存，cdn 缓存，浏览器缓存，服务器缓存。

## 为什么利用多个域名来存储网站资源会更有效？

- CDN 缓存更方便
- 突破浏览器并发限制
- 节约 cookie 带宽
- 节约主域名的连接数，优化页面响应速度
- 防止不必要的安全问题

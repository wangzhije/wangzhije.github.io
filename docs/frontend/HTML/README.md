### [link 标签属性](!./../element/link.md)

- `rel`
  - `stylesheet`：加载 CSS
  - `alternate`：CSS 替换方案；只加载，不渲染
  - `icon`：加载 favicon.ico 图标
    - 用于 PC 端的页签图标，移动设备上用以显示在主屏幕的图标
    - 苹果 IOS 不支持 icon 的解决方案

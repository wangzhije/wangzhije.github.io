# 文档结构

完整的 HTML 文本基本结构

- DTD：Document Type Definition
- html 根元素
- metadata 文档元数据
- body 文档内容
- 注释 `<!-- -->`
- 实体引用：转义字符

HTML 文件扩展名，通常为 `.html` 或 `.htm`

- 用户可以从 Web 服务器中下载
- 并使用 Web **浏览器**来解析和显示

# 基本介绍

> HTML：HyperText Markup Language，超文本标记语言

- 不是一门编程语言，是标记语言/描述语言
- 用来描述网页结构
- 用来告知**浏览器**如何**组织页面**的标记语言
- 指明，标题、段落、文本、超链接、图片、列表、表单
